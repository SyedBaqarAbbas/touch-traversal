# Static asset guide

## Graph artifacts

`data/graph.json`, `data/layouts.json`, `data/manifest.json`, and
`data/pipeline-report.json` are one validated bundle generated by the Python pipeline. Update them
together with:

```bash
make build-graph
```

Do not edit one artifact in isolation. Changes must pass both Python bundle validation and the web
Zod/Graphology tests. Never place real personal-note data here; ignored `private-*` paths are for
local work only and must remain untracked.

## MediaPipe assets

- `models/hand_landmarker/hand_landmarker.task` is the pinned local model.
- `vendor/mediapipe/tasks-vision/wasm/` is the pinned same-origin runtime.

The exact public URLs are part of `lib/hand-worker-protocol.ts`. Do not rename, regenerate, or
upgrade these binaries casually. A deliberate upgrade must update the package/version, protocol
constants, integrity/hash documentation, and worker-protocol tests together.

Do not commit generated test reports or transient browser output under this directory.
